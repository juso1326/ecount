  ✅ 50+ 細粒度權限 (view, create, update, delete, approve 等)
  ✅ 專案級別權限 (owner, manager, member, viewer)

  🎨 使用範例

  租戶管理員操作流程

    1. 登入商家後台
    2. 進入「角色管理」
    3. 建立自訂角色（例如：業務專員）
    4. 勾選需要的權限：
       ☑ projects.view_all
       ☑ projects.create
       ☑ receivables.view
    5. 指派給使用者

  程式碼檢查權限

    // Controller
    if (!$user->can('projects.create')) {
        abort(403, '無權限建立專案');
    }

    // Middleware
    Route::middleware('permission:finances.approve')
        ->put('/receivables/{id}/approve');

    // Blade
    @can('projects.manage_members')
        <button>管理成員</button>
    @endcan
